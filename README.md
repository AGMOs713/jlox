# Jlox

- Lox [tree-walk interpreter](https://en.wikipedia.org/wiki/Interpreter_(computing)#:~:text=Abstract%20syntax%20tree%20interpreter,code%20via%20JIT%20compilation.) made in Java from the free book [*Crafting Interpreters*](https://craftinginterpreters.com) by Robert Nystrom. Made for learning, not to be used.

# Lox

- Lox is a small programming language designed by Robert Nystrom for his book *Crafting Interpreters*. This book is widely recommended for people that want to get into the world of creating programing languages or similar products. You can read it online for free.
